ベッドで寝ると夢の世界を探検できるmodです。

睡眠時にランダムな夢のディメンションに飛ばされます。

# 夢ディメンション

・麦畑(Dream Field)

・アーキアンアース(Ice Burg)

・無限下水道(Sewers)

・水の街路(Street)

・霧の森(Dense Forest)

・深海帯(Abyss)

・地下墓地の通路(Tunnel)

・残橋(Bridge)

・夢ネザー(Dream Nether)

1ダメージでも受けるとプレイヤーは起床します。

# 導入方法

(1.20.1である必要があります)

jarファイルをmodsフォルダにぶち込みます。

前提modはありません。以上です。

# あとがき

実は先駆者兄貴がすでに同じ発想のmodを作っています。残念ながら互換性はないよ。

恐らくこれ系のmodは他にYume Craftしかない。(何ならこのmodが初出だと思っていた)
